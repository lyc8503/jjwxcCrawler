## 晋江文学城小说爬虫

**[2026/02/21] 由于晋江更新, 此项目已经不能用于下载 V 章, 仍能下载免费章节, 下载 V 章请寻找其他方法.**

**本项目仅供学习与交流使用, 下载 TXT 只是用于更方便地阅读, 请支持正版, 不要传播盗版!**

#### 使用方法

1. 下载 Release 中的 exe 文件, 打开运行.
2. clone 本项目, 安装相关依赖后运行 main.py.

依据提示输入相关信息即可, 输出文件为 output.txt.

#### 一些小优点

- 使用 Android API, 比从网页获取信息更加稳定(应该?).
- 输出的 TXT 内容更加详细整洁, 包括了文案, 一句话简介, 每章概要, 卷名称等杂七杂八的内容.
- 能获取被锁章节的标题和内容提要.
